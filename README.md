<h2>Installation</h2>

_Below is how install and setting up app._

#### 1. Clone the repo
   ```sh
   git clone https://github.com/karton92/arkanoid-typescipt
   ```
#### 2. Install NPM packages
   ```sh
   npm install
   ```
#### 3. Run application
   ```sh
   npm start
   ```
